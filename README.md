# A mini project with vue3 x threeJS for Lunar New Year(might be extent for multi purpose)

This is a simple gacha game for drawing lucky money (Asia's culture).

Feature:
  - Draw lucky money(random base on prize's percentage).
  - Setup prize rate.
  - Authenticate with Google.
  - save prize's rate as a profile.

URL: your-lucky-money.vercel.app
